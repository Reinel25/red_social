# Proyecto Red Social

Este proyecto corresponde al desarrollo de la actividad 3, donde se simula la ejecución de un Sprint utilizando la metodología Scrum.

## Descripción
Se desarrolló un prototipo de una red social enfocado en la funcionalidad de mensajería entre usuarios.

## Funcionalidad desarrollada
- Envío de mensajes
- Visualización de conversaciones
- Interfaz de chat

## Evidencias
- Documento del proyecto
- Prototipo del sistema
- Tablero de tareas en Trello

## Metodología
Se utilizó Scrum para la organización del trabajo, incluyendo Product Backlog, Sprint Backlog y seguimiento del Sprint.
